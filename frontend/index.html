⚠️ 设备未配对

当前设备尚未在 OpenClaw Gateway 上完成配对审批，无法建立连接。

请 SSH 登录到 Gateway 服务器，执行以下命令完成配对：

1. 查看待审批设备：

openclaw devices list --json

2. 审批设备（使用上一步 pending 中的 requestId）：

openclaw devices approve <requestId>

💡 或者将已配对电脑的 ~/.openclaw-chat-client/device.json 拷贝到本机相同路径。
